<컴포넌트 구조(section별 설명)>

    - main(Main)
        >> 어린이 식품 정기배송 서비스 첫 화면
        >> App Download 버튼

    - Delivery(정기배송)
        >> 다양한 상품을 정기적으로 배송 할 수 있는 화면

    - Review(고객후기)

        >> Review Slide List 기능으로 정리
        >> App Download 버튼

    - HotMenu(인기상품)
        >> 인기상품 리스트를 정리하여 고객에게 제공
        >> HotTrack Slide List 기능으로 정리
        >> 더보기 Button으로 Link 이동 && App Download 버튼

    - DownIntro(Footer)
        >> Footer && App Download 버튼 & Instagram Link
